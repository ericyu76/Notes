# Spring Data JPA

## Concept

**Spring Data**

*   把不斷重覆的 DAO Coding 簡化, 透過 Repository 就可以產生 CRUD

**Spring JPA (Java Persistant API)**

*   Java EE 的 DAO API
*   需要有  Provider - Hibernate

**JPA Provider (Hibernate)**

*   可以支援 SQL, Non-SQL

**Spring Data JPA**

*   透過 Spring Data 與 JPA 進行整合
